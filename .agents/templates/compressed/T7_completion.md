# T7: Completion Criteria (Compressed Format)

**Savings:** 50% vs checklist format

## Format

```
<done>
  [x] criterion_1
  [x] criterion_2
  [x] criterion_3
  [x] user confirmed
</done>
```

## Rules
- All items use [x] (checked) format
- snake_case descriptions
- Last item should be user confirmation (if applicable)
- Keep to essential criteria only — no verbose explanations
