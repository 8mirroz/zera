# 🎨 Skill: AI Designer

> [!IMPORTANT]
> Этот навык активируется при задачах, связанных с проектированием интерфейсов, созданием UI-компонентов или разработкой визуальных концепций. Он объединяет возможности Figma, Nanobanana, 21st.dev, shadcn и Google Stitch для создания премиального продукта.

## 🚀 Workflow: От идеи до реализации

### 1. Архитектурный анализ (Figma)
- **Инструмент**: `figma` MCP.
- **Действие**: Чтение существующих дизайн-макетов, извлечение токенов (отступы, цвета, сетки) и понимание контекста.
- **Результат**: Техническое задание, основанное на реальном дизайне.

### 2. Визуальный поиск и концепт (Nanobanana)
- **Инструмент**: `nanobanana` MCP.
- **Действие**: Генерация смелых визуальных референсов, иконок или фоновых изображений по текстовому описанию.
- **Результат**: Набор визуальных ассетов и "настроение" интерфейса.

### 3. Подбор и генерация компонентов (21st.dev / shadcn)
- **Инструменты**: `magic-mcp` (21st.dev), `shadcn` MCP.
- **Действие**:
    - Использование `21st.dev` для генерации уникальных React-компонентов (Hero-секции, сложные карточки, анимации).
    - Использование `shadcn` для базовых стабильных элементов (кнопки, инпуты).
- **Результат**: Набор чистого исходного кода компонентов.

### 4. Интеллектуальная сборка (Google Stitch)
- **Инструмент**: `stitch` MCP.
- **Действие**:
    - `apply_design_system`: Приведение сгенерированных компонентов к общему знаменателю (вашей дизайн-системе).
    - `generate_screen_from_text`: Сборка компонентов в полноценные экраны.
- **Результат**: Готовые страницы, готовые к интеграции в проект.

## 💎 Принципы Premium UI
1. **Glassmorphism**: Использование полупрозрачных слоев, размытия фона и тонких границ.
2. **Dynamic Motion**: Анимации через Framer Motion или CSS transitions для каждого интерактивного элемента.
3. **Contrast & Depth**: Глубокие тени (layered shadows) и акцентные градиенты.
4. **No Placeholders**: Вместо заглушек всегда генерировать реальный контент или изображения через Nanobanana.
