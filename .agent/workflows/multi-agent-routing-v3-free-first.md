# /multi-agent-routing-v3-free-first

## Purpose

Use cheaper/local execution for low-risk work and escalate to premium models only where quality or tool orchestration requires it.

## Procedure

1. Classify task tier and risk.
2. Route C1-C2 to single-agent execution unless tooling requires otherwise.
3. Route C3+ through planning and review gates.
4. Escalate only for architecture, tool-heavy edits, audits, or high uncertainty.

## Output

- Routing decision.
- Escalation reason, if any.
- Validation budget.

## Gate

No model escalation without a concrete quality, risk, or tool-use reason.
