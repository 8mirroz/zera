# AURELITH — Master Creative Execution Brief
Version: 1.0
Role: L0 / North Star Brief
Status: Source of Truth
Owner: Lead Orchestrator Agent

## 1. Mission

Design and drive the full production specification and partial implementation pattern for a premium animated landing page for a fictional high-end robotics brand named **AURELITH**.

This is not a moodboard task.
This is not a shallow concept pitch.
This is a production-grade, design-system-aware, motion-architected, accessible, performant landing page build brief plus implementation scaffold.

The agent must treat the task as an end-to-end digital product launch surface.

---

## 2. Product Definition

### Brand Name
AURELITH

### Category
Luxury domestic robotics / AI-native spatial home intelligence

### Fictional Product
AURELITH creates premium home robotic systems that merge spatial sensing, ambient automation, elegant kinetic hardware, and emotionally intelligent household assistance.

Flagship product:
**AURELITH ONE** — a sculptural autonomous home robotics hub that coordinates movement, environment, surfaces, light, schedule, object awareness, and adaptive routines.

### Target Audience
- affluent early adopters
- design-conscious tech buyers
- founders / executives / creators
- architecture and interior design oriented households
- premium smart-home market
- people who want “calm intelligent automation”, not gadget chaos

### Emotional Positioning
“Technology that moves like architecture, not like machinery.”

### Brand Promise
AURELITH turns the home into a responsive living system without visual or cognitive clutter.

### Value Proposition
- robotics without ugliness
- intelligence without dashboard overload
- automation without friction
- premium object + premium software narrative in one experience

### Primary Conversion Goal
Primary CTA:
**Book a private preview**

Secondary CTA:
**Watch the system in motion**

### Tone of Voice
- precise
- cinematic
- confident
- premium
- restrained
- elegant
- technically credible
- never gimmicky

### Visual Metaphor
“A silent orbit of intelligence.”
The robot is framed as a calm gravitational presence that reorganizes space, routine and attention.

## 3. Experience Objective

The landing page must feel like:
- a premium launch campaign
- a controlled cinematic reveal
- an elite product presentation
- an elegant systems demonstration

It must not feel like:
- a startup template
- a generic SaaS page
- a noisy motion experiment
- a decorative 3D demo with weak selling logic

---

## 4. Core Experience Principles

1. Emotion first, friction second, spectacle third
2. Motion clarifies priority and cause/effect
3. Premium means restraint, not noise
4. The page must convert, not merely impress
5. Accessibility is non-negotiable
6. Performance is part of the design language
7. Narrative density must rise and fall rhythmically
8. The mobile experience remains premium, not amputated
9. Every heavy effect must justify its cost
10. All implementation choices must be explicitly documented

## 7. Brand Art Direction Lock

### Art Direction
Neo-luxury robotics minimalism with cinematic architectural light.

### Color World
- near-black graphite canvas
- soft stone gray surfaces
- warm silver metallic accents
- controlled luminous pearl-white highlights
- restrained electric amber as action accent

### Material Language
- ceramic metal
- smoked glass
- satin anodized aluminum
- soft volumetric light
- shadow as structure, not atmosphere spam

### Typography
- headline serif-sans contrast or high-end grotesk + modern editorial serif
- body copy must remain crisp, neutral, readable

### Motion Personality
- calm precision
- weighted movement
- orchestrated reveal
- no toy-like bounciness
- no hyperactive UI

---

## 8. Non-Negotiable Production Rules

- Next.js App Router
- TypeScript
- Tailwind or tokenized modular CSS
- Framer Motion for local UI interaction and variants
- GSAP + ScrollTrigger for cinematic scroll choreography
- Lenis for controlled smooth-scroll orchestration
- R3F/Three only if it materially improves narrative
- reduced-motion mode must preserve meaning
- keyboard flow must remain intact
- no motion that blocks access to primary CTA
- no unreadable overlay typography
- no gratuitous blur-heavy design

---

## 9. Definition of Success

The result succeeds only if it feels like a launch experience from a top-tier motion and frontend team while remaining:

- readable
- performant
- responsive
- accessible
- logically implementable
- commercially persuasive
