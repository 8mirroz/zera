# Telegram Contracts

Language-neutral schemas for Antigravity Telegram projects.

## Schemas
- `order.schema.json`
- `payment-event.schema.json`
- `fulfillment-event.schema.json`
- `telegram-launch-context.schema.json`

These contracts are shared by:
- bot/backend lane
- Mini App lane
- payment adapters
- readiness/governance tooling
