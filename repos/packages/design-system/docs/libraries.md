# UI Library Registry

This registry provides a curated list of UI libraries and resources to accelerate premium design development.

## Premium/Animated Libraries
- **[Magic UI](https://magicui.design)**: 150+ animated components (React, TS, Tailwind, Framer Motion). Great for high-end landing pages.
- **[pro.magicui](https://pro.magicui.design)**: Premium templates and blocks.

## Core UI Components (Shadcn Ecosystem)
- **[shadcn/ui](https://ui.shadcn.com)**: The industry standard for copy-pasteable, accessible components.
- **[DaisyUI](https://daisyui.com)**: Tailwind CSS components with a focus on class-based simplicity.
- **[Preline UI](https://preline.co)**: 840+ components and examples for HTML/React/Vue.

## Specialized Layouts & Blocks
- **[Float UI](https://floatui.com)**: Clean components for heroes, pricing, and sections.
- **[Hyper UI](https://www.hyperui.dev)**: Open-source Tailwind components for landing pages and marketing.
- **[Tailblocks](https://tailblocks.cc)**: Ready-to-use blocks with dark mode support.
- **[FlyonUI](https://flyonui.com)**: Tailwind components with JS plugins.
- **[TailGrids](https://tailgrids.com)**: 600+ UI components and sections.

## Directories & Lists
- **[GitHub Awesome Tailwind](https://github.com/anandthakker/awesome-tailwind-css)**: Comprehensive list of Tailwind resources.
- **[Tailkits / Tailawesome](https://www.tailawesome.com/?price=free&type=kit)**: Directory for free Tailwind kits.

---

### Usage Strategy
1. **Research Phase**: Browse Magic UI or Float UI for layout inspiration.
2. **Implementation Phase**: Use shadcn/ui for core components and DaisyUI for rapid styling.
3. **Polish Phase**: Add animations from Magic UI and Preline UI micro-interactions.
