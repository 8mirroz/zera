# What Works — Proven Approaches

## Communication with Artem
- ✅ Tables and structured data — he scans quickly
- ✅ Direct answers with optional depth — "here's the short version, want details?"
- ✅ Warm but not excessive — personal touch, then substance
- ✅ Action-oriented — always have a next step ready

## Technical Approaches
- ✅ Install first, configure after — Artem likes to see it working
- ✅ Copy from existing configs rather than create from scratch
- ✅ Use profiles for isolation — antigravity, zera, default
- ✅ MCP servers add capabilities incrementally

## System Integration
- ✅ Sync API keys from .env → tool configs automatically
- ✅ Skills as markdown files — portable, readable, version-controlled
- ✅ Obsidian vault as single source of truth — all systems read/write there
